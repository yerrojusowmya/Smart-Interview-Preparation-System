# Smart Interview Preparation System

A Flask-based web application developed to help students prepare for technical and HR interviews.

## Features

- User Registration and Login
- Dashboard
- Python Interview Questions
- Java Interview Questions
- SQL Interview Questions
- HR Interview Questions
- Mock Test System
- Score Calculation
- Resume Upload
- Skill Detection
- ATS Score Generation

## Technologies Used

- Python
- Flask
- HTML
- CSS
- SQLite
- PyPDF2

## Project Objective

The objective of this project is to provide students with a centralized platform to practice interview questions, evaluate their performance, and improve their chances of success in campus placements and job interviews.
